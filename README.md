# 萬用履歷產生器

一個不需要安裝套件、直接用瀏覽器就能使用的履歷製作工具。所有內容都保存在使用者自己的瀏覽器中，不會上傳到伺服器。

## 功能

- 三種 A4 履歷版型：現代雙欄、經典單欄、極簡留白
- 即時履歷預覽與縮放
- 基本資料、專業摘要、工作經歷、學歷、技能、專案、證照、語言、自訂區段
- 動態新增、刪除與調整項目順序
- 主題色、字體密度、繁中／英文標題切換
- 可選擇隱藏照片與聯絡圖示，提升 ATS 相容性
- 瀏覽器自動儲存
- JSON 匯入／匯出備份
- 使用瀏覽器列印功能輸出 A4 PDF
- 響應式介面，支援手機、平板與桌面瀏覽器

## 使用方式

### 直接開啟

下載專案後，直接用瀏覽器開啟 `index.html` 即可。

### 本機伺服器

```bash
python -m http.server 8000
```

接著開啟 `http://localhost:8000`。

## 匯出 PDF

1. 完成履歷內容。
2. 點擊右上角「匯出 PDF」。
3. 在瀏覽器列印視窗中選擇「另存為 PDF」。
4. 紙張選擇 A4、邊界選擇無，並開啟背景圖形。

## 資料隱私

履歷內容會透過 `localStorage` 儲存在目前瀏覽器。專案沒有後端服務，也不會將資料傳送到網路。若要換裝置或清除瀏覽器資料，請先使用「備份 JSON」。

## 專案結構

```text
.
├── index.html                   # 編輯器與預覽頁面
├── styles.css                  # UI、履歷版型與列印樣式
├── app.js                      # 狀態管理、預覽、匯入匯出
└── .github/workflows/pages.yml # GitHub Pages 部署
```

## GitHub Pages

推送到 `main` 分支後，工作流程會自動部署靜態網站。第一次使用時，請到 Repository Settings → Pages，確認來源設為 GitHub Actions。

## 開發原則

此專案採用原生 HTML、CSS 與 JavaScript，不依賴前端框架或外部 CDN，因此可離線使用，也容易部署到任何靜態網站服務。
